# writing-toolkit

A team of Claude Code skills for writing with a distinctive **voice**: *actual* (modern),
*amena* (engaging), *sutil* (subtle), and *punzante* (sharp/incisive) when the point earns
it — always clear first. Output in Spanish by default.

## Skills included

| Skill | What it does |
| :--- | :--- |
| `escritura` | Write/rewrite any text (articles, essays, posts, emails, scripts) in the voice |
| `copywriting` | Persuasive marketing/sales copy that converts (landing, ads, CTAs, emails) |
| `contenido-social` | Social posts and threads (LinkedIn, X, Instagram) with scroll-stopping hooks |
| `editor-estilo` | Edit/polish a draft: tighten, sharpen, fix rhythm, cut filler — and show what changed |

Pairs with `seo-landing` (copy for landing pages) and `report-builder` (clear reports).

## The voice
- **Actual y natural** — contemporary, no corporate stiffness.
- **Amena** — warm, with rhythm; a pleasure to read.
- **Sutil** — intelligence suggested, not shouted.
- **Punzante solo si suma** — a sharp, memorable line when it serves; never gratuitous, never
  cruel to people (the edge goes to ideas).
- **Claridad ante todo** — if a word doesn't earn its place, it's cut.
